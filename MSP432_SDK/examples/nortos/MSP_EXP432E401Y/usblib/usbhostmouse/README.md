# usbHostMouse

---

## Example Summary

 This example application demonstrates how to support a USB mouse using
 the MSP_EXP432E401Y evaluation kit.  This application supports only a
 standard mouse HID device.

 UART0, connected to the Virtual Serial Port and running at 115,200, 8-N-1,
 is used to display messages from this application.

## Example Usage

>__Important:__ Adding the define USE_ULPI to the pre-processor defines allows the use of an external USB High Speed PHY.

Detailed Description

CAN_STATUS_* macros are general status codes returned by CAN_control()

Macro Definition Documentation

§ CAN_STATUS_SUCCESS

#define CAN_STATUS_SUCCESS   (0)

Successful status code returned by CAN_control().

CAN_control() returns CAN_STATUS_SUCCESS if the control code was executed successfully.

§ CAN_STATUS_ERROR

#define CAN_STATUS_ERROR   (-1)

Generic error status code returned by CAN_control().

CAN_control() returns CAN_STATUS_ERROR if the control code was not executed successfully.

§ CAN_STATUS_UNDEFINEDCMD

#define CAN_STATUS_UNDEFINEDCMD   (-2)

An error status code returned by CAN_control() for undefined command codes.

CAN_control() returns CAN_STATUS_UNDEFINEDCMD if the control code is not recognized by the driver implementation.

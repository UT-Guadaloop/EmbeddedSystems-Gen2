Detailed Description

MSP432E4 ADCBuf_Params Parameter Extensions.

To use non-default MSP432E4 specific parameters when calling ADCBuf_open(), a pointer to an instance of this structure must be specified in ADCBuf_Params.custom. Alternatively, these values can be set using the ADCBuf_control() function after calling ADCBuf_open().

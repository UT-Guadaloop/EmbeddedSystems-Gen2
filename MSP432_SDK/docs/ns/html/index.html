Network Services API Reference

Modules

The Network Services Component is a portable networking API that abstracts different network stacks found on TI devices.

  • SlNetSock Overview

Network Services also includes support for the following application layer networking protocols:

  • HTTPClient - Provides a HTTP client interface to connect to, and communicate with, HTTP servers.
  • HTTPServer - Provides a HTTP server interface to instantiate an HTTP server, and handle requests from HTTP clients.
  • MQTTClient - Provides a MQTT client interface to connect to, and communicate with, MQTT servers.
  • MQTTServer - Provides a MQTT server interface to instantiate a MQTT broker, and handle requests from MQTT clients.
  • SNTP Client - Provides a SNTP interface to synchronize local time with NTP servers.

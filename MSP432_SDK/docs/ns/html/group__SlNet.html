int32_t ti_net_SlNet_initConfig ( )

Prototype for SysConfig generated initialization function that initializes various parts of the NS stack based on the user's SysConfig script. This function should be called after the lower network stacks initialize. Non-SysConfig users can provide their own implementation of this function in their application's source code.

Returns
Zero on success, or negative error code on failure
